Welcome to the Komodo Tech Pypiserver!

This is a PyPI compatible package index serving 1 packages.

To use this server with pip, run the the following command: 

pip install --extra-index-url https://pypi.komodotech.io/ PACKAGE [PACKAGE2...]

To use this server with easy_install, run the the following command: 

easy_install -i https://pypi.komodotech.io/simple/ PACKAGE

To upload a package, run the following commands: 

python3 setup.py sdist bdist_wheel
twine upload --repository-url https://pypi.komodotech.io dist/*

The complete list of all packages can be found here or via the simple index.

This instance is running version 1.2.7 of the pypiserver software.
